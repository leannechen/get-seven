# Get Seven

## Description

Let g(N) be the count of numbers that contain a 7 when you write out all the numbers from 1 to
N.
For example:

- g(7) = 1
- g(20) = 2
- g(70) = 8
- g(100) = 19

## Scripts
1. `yarn test`: Run tests for codes
1. `yarn test-watch`: Watch for changes and run tests accordingly
